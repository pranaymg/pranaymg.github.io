# Personal Finance Tracker

Personal Finance Tracker is a modern web application that helps you manage your finances effectively. With this application, you can track your income and expenses, categorize transactions, and gain better control over your financial health.

## Key Features Implemented

### UI/UX Enhancements
- iOS-style liquid glass effect with blur(40px), saturate(200%), brightness(110%)
- Animated shine effects on cards
- Multiple inset shadows for depth
- Centered login/signup forms with glassmorphism
- Modern input fields with focus animations
- Mobile-optimized button sizes and spacing

### Functionality
- Category tracking (Food, Shopping, Transport, Bills, Entertainment, Health, Education, Salary, Other)
- Category analysis modal with gradient progress bars
- SMS transaction parser
- Real-time balance calculation
- Transaction CRUD operations
- Data sync with JSONBin.io

### Theme System
- Dark theme (default)
- Light theme
- Theme persistence across sessions
- Toggle button in navbar
